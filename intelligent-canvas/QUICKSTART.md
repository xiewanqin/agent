# 快速启动指南

## 1. 安装依赖

```bash
cd intelligent-canvas
npm install
```

## 2. 配置环境变量

创建 `.env` 文件（参考 `.env.example`）：

```env
VITE_OPENAI_API_KEY=sk-your-api-key-here
VITE_OPENAI_BASE_URL=https://api.openai.com/v1
VITE_MODEL_NAME=gpt-4
```

**注意**：如果你使用的是其他LLM服务（如通义千问、文心一言等），需要修改 `VITE_OPENAI_BASE_URL` 为对应的API地址。

## 3. 启动开发服务器

```bash
npm run dev
```

浏览器会自动打开 http://localhost:3000

## 4. 使用说明

### 添加节点
1. 从左侧节点面板拖拽节点类型到画布
2. 或使用AI工具栏输入需求，点击"智能推荐"

### 连接节点
1. 点击节点的输出端口（底部），拖拽到目标节点的输入端口（顶部）
2. 或使用"智能连线"功能自动连接

### AI功能
- **智能推荐**：在AI工具栏输入框输入你的需求，点击"智能推荐"按钮
- **智能布局**：点击"智能布局"按钮，AI会自动优化节点位置
- **智能连线**：点击"智能连线"按钮，AI会根据节点类型推荐连接关系

### 性能优化
- 点击右上角性能监控面板中的下拉菜单
- 选择"高性能模式"以启用Web Workers和虚拟滚动
- 实时查看FPS、渲染时间等性能指标

## 5. 常见问题

### Q: AI功能无法使用？
A: 请检查 `.env` 文件中的API配置是否正确，确保API密钥有效。

### Q: 性能较差？
A: 尝试切换到"高性能模式"，或减少画布上的节点数量。

### Q: 节点无法拖拽？
A: 确保从左侧节点面板拖拽，而不是点击。
